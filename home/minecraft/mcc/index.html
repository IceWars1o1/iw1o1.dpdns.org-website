Termux 安装 Minecraft Console Client 教程

一、 准备工作：

  • 获取官方版的 Termux 或 ZeroTermux 软件，并完成安装和初始化
  • 能正常联网的手机

二、安装步骤：

  1. 更新软件包
    • 输入以下内容换源：
    shell 
    termux-change-repo

    在界面中点击按键栏的下键（↓），按下键盘的空格键（└┘），按下键盘或按键栏的回车键（←┘）。然后再在接下来的界面中使用按键栏的上（↑）下（↓）键控制强调行至 ( ) mirrors.tuna.tsingMirror by Tsinghua University TUNA Association 一行，并按下空格（└┘）键，然后按下回车（←┘）键。等待代码跑完，执行下一步。

  2. 更新依赖

    输入以下内容以更新依赖：

    shell 
    pkg update && pkg upgrade -y

    等待代码跑完执行下一步。

  3. 安装容器包及需要的软件包

    输入以下内容安装 proot-distro：

    shell 
    pkg install proot-distro -y && pkg install vim -y

    等待代码跑完执行下一步。

  4. 安装容器

    双指在控制台捏合，缩小画面尺寸。

    先输入

    shell 
    vim $PREFIX/etc/proot-distro/ubuntu.sh

    进入文本编辑器。按下键盘 i 键，然后通过按键栏的上下左右键移动光标，修改 3 个 TARBALL_URL 行 = 后面的链接：在每个 https:// 前加上：

    text 
    https://gh-proxy.com/

    此步骤可以显著提高下载速度。三行的内容都修改完成后，按下按键栏的 Esc 键，然后输入 :wq 保存并退出。

    然后输入

    shell 
    proot-distro install ubuntu

    等待下载安装步骤的完成。完成后，不出意外会出现如下输出：

    text 
    [*] Finished.

Log in with: proot-distro login ubuntu
  5. 进入容器

    键入以下内容：

    shell 
    proot-distro login ubuntu

    稍等片刻，命令行标记会从 ~ $ 变为 root@localhost:~#。此时，你已成功进入 Ubuntu 24.04 系统。

  6. 更新依赖（可选：换源）
    shell 
    cp /etc/apt/sources.list.d/ubuntu.sources /etc/apt/sources.list.d/ubuntu.sources.bak

cat > /etc/apt/sources.list.d/ubuntu.sources <<'EOF'
Types: deb
URIs: http://mirrors.tuna.tsinghua.edu.cn/ubuntu-ports/
Suites: noble noble-updates noble-security noble-backports
Components: main restricted universe multiverse
Signed-By: /usr/share/keyrings/ubuntu-archive-keyring.gpg
EOF

apt update
apt upgrade -y

    容器系统独立于 Termux 终端。键入以下命令：

    shell 
    apt update && apt upgrade -y

    此次等待时间可能较长，具体时间取决于网络环境。

  7. 安装 .NET 8

    输入：

    shell 
    apt install dotnet-sdk-8.0 -y

    跑完后限制其内存申请最大值：

    shell 
    echo 'export DOTNET_GCHeapHardLimit=200000000' >> ~/.bashrc
echo 'export DOTNET_EnableDiagnostics=0'        >> ~/.bashrc
source ~/.bashrc

    验证安装：

    shell 
    dotnet --version
  8. 获取 Minecraft Console Client

    键入以下命令：

    shell 
    mkdir mcc
cd mcc

    然后使用系统浏览器访问网页 Minecraft Console Client，按照你的 CPU 架构找到合适的 Linux 版本。

    以下以 CPU 架构为 Arm64(AARCH64) 的系统为例，复制对应文件的地址，然后在运行着 proot 容器的 Termux 里输入：

    shell 
    wget https://gh-proxy.com/[你复制的地址] -O MinecraftClient

    例如：

    shell 
    wget https://gh-proxy.com/https://github.com/MCCTeam/Minecraft-Console-Client/releases/download/20250522-285/MinecraftClient-20250522-285-linux-arm64 -O MinecraftClient

    等待下载完成后输入：

    shell 
    chmod +x MinecraftClient
  9. 运行爽玩！
    shell 
    ./MinecraftClient

    完成后，可参照 @芒果方块 的教程使用！
